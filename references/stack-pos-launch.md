# Stack Pós-Launch v6.0

## Nível 1 — Essencial (todo projeto)

| Ferramenta | Função | Custo |
|------------|--------|-------|
| **GA4** | Tráfego, conversões, eventos, funis | Gratuito |
| **Search Console** | Indexação, keywords, erros de rastreamento | Gratuito |
| **Clarity** (Microsoft) | Heatmaps, gravações de sessão | Gratuito |

### Configuração Mínima GA4
- Eventos: `page_view`, `generate_lead`, `form_submit`
- Conversões: marcar `generate_lead` como conversão
- Funis: Landing Page → Formulário → Obrigado

### Configuração Clarity
- Instalar script no `<head>`
- Habilitar gravação de sessões
- Configurar heatmaps para páginas principais

## Nível 3 — Especializado (e-commerce, alto volume)

| Ferramenta | Função | Custo |
|------------|--------|-------|

## Quando Usar Cada Nível

| Cenário | Nível |
|---------|-------|
| LP simples, cliente pequeno | 1 (GA4 + Console + Clarity) |
| Site institucional, budget moderado | 1 + 2 (+ Hotjar + Looker) |
| E-commerce, alto volume de tráfego | 1 + 2 + 3 (completo) |
