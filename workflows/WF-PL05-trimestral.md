# WF-PL05 — Auditoria Trimestral

**Trigger:** 3 meses após launch, recorrente
**Agente:** Otimizador Pós-Launch (20) + Auditor (18)

## Escopo da Auditoria

### SEO
- Indexação no Google (Search Console)
- Keywords rankeando vs planejadas
- Backlinks novos
- Erros de rastreamento

### Performance
- Core Web Vitals (evolução 3 meses)
- PageSpeed Score (mobile + desktop)
- Tempo de carregamento real (GA4)

### Segurança
- SSL válido e atualizado
- Headers de segurança presentes
- .env inacessível
- Dependências atualizadas

### Conteúdo
- Copy ainda relevante?
- Depoimentos atualizados?
- Informações de contato corretas?
- Preços e ofertas atuais?

## Relatório Trimestral
- Evolução de métricas (gráficos se possível)
- ROI estimado (conversões × valor médio)
- Comparativo trimestre anterior
- Recomendações estratégicas (próximos 3 meses)
