# Referências e Benchmark — Portfólio Sonbatera v6.0

## CABEÇALHO
- **Projeto:** sonbatera-portfolio
- **Agente:** Caçador (02.1)
- **Data:** 2026-05-02
- **Nicho:** Portfólio Artístico / Música

## FASE 0 — DESCONSTRUÇÃO DE INSPIRAÇÕES
### 1. Aaron Spears / Matt Halpern (Portfólios de Bateristas Globais)
- **Dimensão Visual:** Predominância de imagens monocromáticas com destaques para luzes de palco neon (vermelho/azul). Minimalismo, foco total na bateria.
- **Dimensão Comunicação:** "Let the playing speak". O vídeo é mais importante que o texto.
- **Dimensão Negócio:** Redirecionamento rápido para agenciamento/booking.

### 2. Awwwards / Lapa Ninja (Dark Mode Music Portfolios)
- **Layouts "One Page":** Rolagem vertical suave (smooth scroll).
- **Tipografia:** Uso de fontes condensadas pesadas (Oswald, Bebas Neue ou Tungsten) para cabeçalhos para transmitir "Pancada/Batida", e fontes sans-serif finas para o corpo (Montserrat/Inter).

## SEÇÃO 1 — BANCO DE REFERÊNCIAS VISUAIS
- Efeitos de luz no fundo (`radial-gradients`).
- Elementos gráficos simulando frequências sonoras ou ondas rítmicas nas transições de seção.
- Botões de "Play" (YouTube) como componente central (overlay).

## SEÇÃO 2 — OPORTUNIDADE DE DESIGN
- Transformar a LP numa experiência imersiva de show. O fundo preto absoluto (Dark Mode verdadeiro `#000000` ou `#0B0B0B`) vai fazer os vídeos e as luzes de palco nas thumbnails pularem na tela.
